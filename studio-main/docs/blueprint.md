# **App Name**: Confir NSPS

## Core Features:

- Registration Form: Collect user information via a comprehensive and user-friendly form.
- User Authentication: Securely authenticate users via email and password.
- Profile Management: Allow users to manage and update their profiles.
- Confirmation: Automatically confirm their NSPS status using internal/external tools.
- Admin Dashboard: Provide administrators with a dashboard to manage users and system settings.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) to convey trust and stability.
- Background color: Light gray (#F5F5F5) for a clean, modern look.
- Accent color: Teal (#009688) for highlights and interactive elements.
- Body font: 'Inter', a grotesque-style sans-serif for a modern, objective look; suitable for body text
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel; suitable for headlines
- Use clear, minimalist icons from a consistent icon set.
- Employ a clean, grid-based layout for easy navigation and readability.
- Use subtle animations for transitions and feedback.
